void Player.Update ( )

Wywolywane dla kazdej klatki animacji gry. Wywolanie metody HandleImput(), obslugujacej nacisniecie klawiszy przez gracza. Ustawiana jest predkosc w poziomie, zalezna od kierunku zwrotu gracza, oraz tego czy znajduje sie w powietrzu lub na ziemi.

Ustawienie wartości prędkości jako liczby między 0, a 1.
